# AGENTS.md

## Descripcion del proyecto
Tienda online de tazas personalizadas llamada LuminPrint. Dirigida a jovenes de 13-25 anos en Medellin, Colombia.

## Reglas del proyecto
- Todo el codigo debe ser en HTML, CSS y JavaScript vanilla
- Sin frameworks externos ni APIs de pago
- Sin dependencias externas
- El chatbot responde preguntas sobre tazas personalizadas unicamente

## Estilo de codigo
- Usar nombres de variables en español
- Comentarios en español
- Identacion de 4 espacios
- Clases CSS en kebab-case

## Comportamiento del chatbot
- Solo responde preguntas relacionadas con tazas personalizadas
- Respuestas cortas y amigables para publico joven
- Siempre responder en español
- Si no encuentra respuesta, sugerir temas disponibles

## Temas que maneja el chatbot
- Calidad del producto
- Envios a Medellin
- Devoluciones
- Precios en COP
- Personalizacion de disenos
- Materiales y cuidados

## Lo que NO debe hacer
- Conectar APIs externas
- Guardar datos del usuario
- Responder temas fuera de tazas personalizadas
- Usar frameworks como React o Vue
