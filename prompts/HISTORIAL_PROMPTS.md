# Historial de Prompts - LuminPrint

## Etapa 1: Definicion del problema
**Prompt:** Ayudame a buscar un proyecto para poder ejecutar esto: construir un parrafo con la problematica y la solucion propuesta, ampliar la problematica apoyado con IA, definir criterios de aceptacion, construir las fases del proyecto y construir codigo.
**Por que lo use:** Para definir el tipo de proyecto que queria desarrollar.

## Etapa 2: Seleccion del negocio
**Prompt:** Vendo tazas de mugs personalizadas, publico joven 13-25 anos, precio bajo 5-30 USD, solo Colombia-Medellin.
**Por que lo use:** Para darle contexto especifico al proyecto y que las respuestas fueran relevantes para mi negocio real.

## Etapa 3: Ampliacion de la problematica
**Prompt:** Busca datos especificos sobre dudas de clientes en tiendas de tazas personalizadas y el impacto del abandono de carrito.
**Por que lo use:** Para respaldar la problematica con datos reales e investigacion.

## Etapa 4: Criterios de aceptacion
**Prompt:** Define criterios de aceptacion especificos para el chatbot de tazas personalizadas con metricas medibles.
**Por que lo use:** Para tener objetivos claros y medibles del proyecto.

## Etapa 5: Diseno visual
**Prompt:** Quiero colores morados y rojos, degradados, muy bien disenados, moderno y minimalista. El nombre de la marca es LuminPrint.
**Por que lo use:** Para definir la identidad visual antes de codificar.

## Etapa 6: Desarrollo del codigo
**Prompt:** Dame el codigo HTML/CSS completo para la interfaz de LuminPrint con chatbot flotante interactivo.
**Por que lo use:** Para generar el codigo base de la interfaz que luego ajuste manualmente.

## Etapa 7: Logica del chatbot
**Prompt:** Crea la logica del chatbot en JavaScript con base de datos de preguntas frecuentes sobre tazas personalizadas.
**Por que lo use:** Para implementar las respuestas automaticas sin usar APIs externas.

## Anotaciones generales
- Se itero varias veces el diseno hasta lograr el resultado deseado
- Se simplificaron varias propuestas de la IA que eran demasiado complejas
- Se ajustaron precios a pesos colombianos (COP)
- Se elimino la idea de conectar APIs externas por simplicidad
